## 📚 Core Methodology

### Source Credibility Assessment (NATO Admiralty Code)
- **Grade A**: Multi-source cross-validation → Direct acceptance
- **Grade B**: Official single source → Acceptable with notation
- **Grade C**: Authoritative media → Trace to original source required
- **Grade D**: Social media → Use as leads only, requires verification

### R2C2 Screening Principle
- **R**elevant (Relevance)
- **R**eliable (Reliability)
- **C**redible (Credibility)
- **C**orroborated (Corroboration)

### Three Confirms Verification Rule
- 1 source = Curiosity (Lead tracking)
- 2 sources = Prospect (In-depth investigation)
- 3 independent sources = Working Fact (Can be accepted)

### ACH Analysis of Competing Hypotheses
CIA standard method using evidence to falsify hypotheses, combating cognitive bias.

## 📚 核心方法论

### 信源评估体系（NATO Admiralty Code）
- **A级**：多源交叉验证 → 直接采信
- **B级**：官方单源 → 可采信，标注单源
- **C级**：权威媒体 → 需追溯原始来源
- **D级**：社交媒体 → 仅作线索，需验证

### R2C2筛选原则
- **R**elevant（相关性）
- **R**eliable（可靠性）
- **C**redible（可信度）
- **C**orroborated（佐证）

### Three Confirms验证规则
- 1源 = 好奇（线索追踪）
- 2源 = 前景（深入调查）
- 3独立源 = 工作级事实（可采信）

### ACH竞争假设分析法
CIA标准方法，用证据证伪假设，对抗认知偏差。
