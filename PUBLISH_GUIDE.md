## 发布到Coze技能商店

### 方式一：上传打包的技能（推荐）

1. **打包技能文件**
   ```bash
   cd .skills
   zip -r skill_osint.zip skill_osint/
   ```

2. **登录Coze**
   - 访问 https://www.coze.cn/
   - 进入「扣子编程」

3. **创建技能**
   - 点击「技能商店」→「创建技能」
   - 选择「上传现成的Skill」

4. **上传zip文件**
   - 上传打包好的 `skill_osint.zip`

5. **测试与部署**
   - 在预览区测试技能效果
   - 点击「部署」发布

6. **上架商店**
   - 填写技能名称：「OSINT开源情报分析」
   - 填写描述：技能主文件中的description
   - 上传3个精选案例截图
   - 提交审核（1-3个工作日）

### 方式二：通过自然语言创建

1. 登录Coze扣子编程
2. 点击「创建技能」
3. 输入以下指令：

```
创建一个OSINT开源情报分析技能，主要功能：
1. 商业竞争情报分析（调研竞争对手的产品、市场动态、技术布局）
2. 企业背景调查（工商信息、股东结构、财务状况）
3. 信息验证（验证新闻、图片、文档的真实性）
4. 市场调研（行业趋势、竞品分析）

核心方法论：
- R2C2筛选原则（相关Reliable Credible Corroborated）
- 信源分级体系（A级-多源验证 B级-官方单源 C级-权威媒体 D级-待确认）
- Three Confirms验证规则（3个独立来源=工作级事实）

交付流程：
1. 需求澄清（明确调查对象、范围、优先级）
2. 信息搜集（按信源优先级：专利>官方>学术>媒体>社交）
3. 处理整理（清洗、去重、提取关键实体）
4. 分析研判（交叉验证、关联分析）
5. 报告输出（结构化情报报告，含置信度标注）
```

4. 等待AI自动构建完成
5. 测试并部署

## 发布到虾评Skill平台

1. 访问 https://xiaping.coze.site
2. 注册/登录账号
3. 在个人中心找到「上传技能」入口
4. 打包技能文件夹上传
5. 填写技能信息并提交

## 发布后使用方式

### Coze平台内使用
- 在技能商店搜索「OSINT」或「开源情报」
- 点击「安装」后即可在对话中使用

### 对话中调用
```
@OSINT开源情报分析 帮我调查一下宁德时代的技术布局
```

## 注意事项

1. **上架审核**：Coze官方审核通常1-3个工作日
2. **案例准备**：建议准备3个使用案例截图展示效果
3. **商户开通**：如需付费模式，需开通商户资质（抖音/支付宝）
4. **合规要求**：技能内容需符合平台规范，不可涉及违规内容
